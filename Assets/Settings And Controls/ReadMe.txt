To use, Simply create an empty gameobject and apply the "Settings and Controls" script to it. You then need to put this gameObject in the first scene your game loads up. It will be automatically replicated through any other scene that is loaded and can be accessed statically.

Look through the example scripts for help on using the system. All classes and functions have been docutmented to help you.
